## Contexto del Repositorio del Proyecto en GitHub

**Objetivo:** Guardar y versionar el proyecto `predicador` en un repositorio remoto.

**Plataforma Seleccionada:** GitHub

---

### Información del Repositorio

*   **URL del Repositorio:** `https://github.com/williamcmm/Predicadores-cat-licos.git`
*   **Rama Principal:** `main`
*   **Remoto Configurado como:** `origin`

---

### Proceso de Actualización (Manual)

Las actualizaciones al repositorio remoto se deben realizar únicamente a solicitud del usuario.

Los pasos para una actualización son:

1.  **Añadir cambios al área de preparación:**
    `git add .`

2.  **Crear un commit con un mensaje descriptivo:**
    `git commit -m "Tu mensaje descriptivo aquí"`

3.  **Subir los cambios al repositorio remoto:**
    `git push origin main`
